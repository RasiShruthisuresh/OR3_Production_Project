# Olivia Rodrigo Production Evolution — Data Engineering Project

Analyzing how Olivia Rodrigo's production quality and audio characteristics evolved across her
three studio albums (*SOUR*, *GUTS*, *you seem pretty sad for a girl so in love*), and whether
that evolution correlates with audience engagement. Built for a Medium article combining data
engineering and pop culture analysis.

> **Status:** project scaffold in progress. This README is a placeholder and will be replaced
> with the full write-up (pipeline explanation, findings, caveats, reproduction steps) once the
> notebook is complete.

## Data sources

- Track metadata and popularity scores: [Spotify Web API](https://developer.spotify.com/documentation/web-api)
- Tempo/key data: [GetSongBPM](https://getsongbpm.com)
- Chart positions: compiled from Wikipedia
- Remaining audio features: manually compiled (see notebook for methodology)
